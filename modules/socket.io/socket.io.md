# socket.io 是什么
类似 socket 的长连接通讯。可以用它编写类似聊天室之类的应用。
解决了 http 这种单工无状态协议的如下问题。
1. 实现了消息推送功能
2. 通讯服务器的状态监控
3. 稳定可靠的消息连接
4. 支持二进制数据流的发送
5. 多方通讯的消息同步
