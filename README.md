概述
===

**前言:讲解 node 在实际开发中的各方面知识**

---

## 目录结构
* `api`
    node 原生 API 学习,基于 [node 官方文档](https://nodejs.org/dist/latest-v8.x/docs/api/documentation.html) 讲解 node 使用
* `modules`
    讲解基于 npm 包管理系统的各种模块学习,包括 npm 本身
* `practice`
    讲解使用 node 进行各种开发的最佳实践
* `theory`
    深挖 node 原理 
    
所有示例及文档编写遵循以下原则

* 一起以实用为主
* 模块讲解以场景举例,深挖模块原理
